# Install MongoDB Ubuntu 1804

This bash script automates the installation of MongoDB on Ubuntu 18.04 LTS. It updates the system, installs the MongoDB package, and ensures the service is running and enabled on boot. The script concludes by showing how to verify the service status.

## Language

`bash`

## How to Run

Save the code as install_mongodb.sh, make it executable with 'chmod +x install_mongodb.sh', and run it with 'sudo ./install_mongodb.sh'.
